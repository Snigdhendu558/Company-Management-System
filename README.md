# Company Management System

## Overview
A SQL-based Company Management System developed using MySQL to manage employees, departments, projects, and salaries.

## Features
- Employee Management
- Department Management
- Project Management
- Salary Management
- SQL Joins
- Views
- Stored Procedures
- Triggers
- ER Diagram

## Database Tables
- Employees
- Departments
- Projects
- Salaries
- Employee_Projects

## Technologies Used
- MySQL
- MySQL Workbench
